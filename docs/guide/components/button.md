# Button 按钮
  常用的操作按钮。

::: spoiler 使用`type`、`plain`、`round`和`circle`属性来定义99k的样式

```vue
<c-button>Default Button</c-button>

<c-button>Default Button</c-button>
```
:::


### 属性

| 名称 | 参数 | 描述 |
| ------ | ------ | ------ |
| width |  | 按钮宽度 |



### 事件
| 事件 | 参数 | 描述 |
| ------ | ------ | ------ |
| click |  | 鼠标点击事件 |
